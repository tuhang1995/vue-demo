# vue-demo2

## Project setup

```
npm install
```

### 启动 json-server

```
 json-server --host 192.168.0.54 db.json
 可以不写ip 访问就是本地localhost
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### 文件目录

```
  page-base   //组件目录
    animationDamo.vue   //模态框动画效果
    index.vue           //主入口
    merge-table.vue     //合并表格 element
    nowVideo.vue        //在线视频播放
```
