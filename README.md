### 模板介绍

1. common 文件夹下封装了 axios 方法， 如果你有更好的封装方法，可用自己的
2. api 文件是接口文件，统一管理
3. asstes 静态资源
4. components 全局公用组件
5. filters 过滤器
6. lib 外部引用的插件存放及修改文件
7. common 通用的一些文件 包括 css、js
8. views 视图目录
